Vite commands:
  Creae project: npm create vite@latest
  Start project: npm run dev
  Build project: npm run Build
  Preview project: npm run Preview
  Build with specific configurations (.env): npm run build -- --mode [config_name]

  Bundle Visualizer
    npx vite-bundle-visualizer
    npx vite-bundle-visualizer -t network
  
  Host mode (for using external devices)
    vite --host
    vite preview --host
